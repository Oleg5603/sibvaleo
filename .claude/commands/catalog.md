# /catalog

Показывает каталог препаратов Siberian Wellness из базы с фильтрацией.

## Использование
```
/catalog                    # все препараты
/catalog --stage 1          # только ОЧИЩЕНИЕ
/catalog --category суставы # по категории
/catalog --search омега     # поиск по названию
```

## Что делает
Читает `assets/data/products.json` и выводит таблицу:
| Название | Серия | Этап | Категория | Доза взрослым | Курс |
|----------|-------|------|-----------|---------------|------|

## Этапы
- 1 ОЧИЩЕНИЕ
- 2 ЗАЩИТА  
- 3 ПИТАНИЕ
- 4 ВОССТАНОВЛЕНИЕ
