# /build-program

Генерирует 60-дневную текстовую программу для клиента на основе введённых данных.
Выводит готовый текст который можно скопировать и отправить клиенту.

## Использование
```
/build-program
```
Затем следует ввод клиентских данных в диалоговом режиме:
- Имя, возраст, пол
- Жалобы и симптомы
- Диагнозы
- Результаты анализов

## Что делает
1. Читает базу препаратов
2. Матчит по жалобам
3. Строит последовательность: Этап1 → +3дня → Этап2 → +3дня...
4. Не более 3 препаратов одновременно
5. Выводит:
   - Таблицу программы (День / Препарат / Доза / Кратность)
   - Рекомендуемые анализы ДО начала курса
   - Правила приёма

## Правила построения программы
- Новый препарат — через 3-4 дня после предыдущего
- Максимум 3 препарата одновременно
- Порядок: Очищение (1) → Защита (2) → Питание (3) → Восстановление (4)
- Дозировка по возрасту: до 6 лет — не рекомендовано, 6-12, 12-18, взрослые 18-65, пожилые 65+
