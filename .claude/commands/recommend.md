# /recommend

Быстрый подбор препаратов Siberian Wellness по жалобе или диагнозу без открытия приложения.

## Использование
```
/recommend [жалоба или диагноз]
```

### Примеры
```
/recommend гастрит изжога вздутие
/recommend суставы боли в колене остеоартроз
/recommend выпадение волос щитовидная гипотиреоз
/recommend после антибиотиков дисбактериоз
```

## Что делает
1. Читает `assets/data/products.json` и `assets/data/conditions.json`
2. Матчит симптомы/диагнозы с показаниями препаратов
3. Выдаёт топ-5 подходящих препаратов с дозировкой
4. Показывает рекомендуемые анализы
5. Предлагает логическую последовательность по этапам

## Формат ответа
```
ЭТАП 1 — ОЧИЩЕНИЕ:
  • [Препарат] — [доза] — [курс]

ЭТАП 2 — ЗАЩИТА:
  • [Препарат] — [доза] — [курс]

...

АНАЛИЗЫ: [список рекомендуемых]
```
