# Sibvaleo — Инструкция для консультанта

## Как начать работу

### Шаг 1 — Добавить клиента
Нажмите кнопку **«+ Новый клиент»** на главном экране.

**Поля формы:**

| Поле | Что вводить |
|---|---|
| **Имя** | Имя или псевдоним клиента |
| **Возраст** | Полных лет |
| **Пол** | М / Ж |
| **Вес (кг)** | Текущий вес |
| **Рост (см)** | Рост клиента |
| **Основная цель** | Похудение / Иммунитет / Энергия / Детокс / Суставы / Другое |

### Шаг 2 — Симптомы и жалобы

Программа предлагает **9 категорий быстрого скрининга** — отметьте галочками что беспокоит:

| Категория | Примеры жалоб |
|---|---|
| Пищеварение | Запоры, вздутие, тяжесть после еды |
| Печень / детокс | Кожные высыпания, аллергии, приём лекарств |
| Иммунитет | ОРВИ 3+ раз в год, долгое восстановление |
| Витамины/минералы | Выпадение волос, ломкость ногтей, судороги |
| Усталость / стресс | Слабость, нарушение сна, тревога |
| Суставы и кости | Боли в суставах, хруст, остеохондроз |
| Сердце и сосуды | Повышенное давление, варикоз, отёки |
| После болезни | После ОРВИ, антибиотиков, операции |
| Антивозрастная | Кожа, память, обмен веществ |

> **Совет:** можно дополнительно ввести симптомы вручную в поле «Дополнительные жалобы».

### Шаг 3 — Программа

После заполнения нажмите **«Составить программу»**.

Программа разбита на **4 этапа по 60 дням**:
- **Этап 1** (дни 1–15): очищение
- **Этап 2** (дни 16–30): восстановление
- **Этап 3** (дни 31–45): укрепление
- **Этап 4** (дни 46–60): поддержка

### Экспорт

Готовую программу можно сохранить:
- **PDF** — для печати или отправки клиенту
- **HTML** — открывается в браузере

## Частые вопросы

**Почему нет рекомендаций?**
Убедитесь что отмечен хотя бы один симптом или цель.

**Как изменить программу?**
Откройте клиента → нажмите «Редактировать» → пересоставьте программу.

**Программа сохраняется?**
Да — все клиенты и программы хранятся локально на вашем устройстве.
